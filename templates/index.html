Wire Transfer Management System (Class Project)

This project is a C++ command-line program that parses through bank info in .txt format and uses the information to analyze excution dates, sender amounts, bank fees, etc. Made use of priority queues to execute transactions in order of dates provided. I also used Hash Tables so store custom user struct data such as user-name and sets within objects of the user struct to store IP Address information. With the data provided, I also created multiple queuries to analyze information for customers and the bank. This project helped better Data Structures knowledge as well as getting comfortable with the command line (input/output redirection and creating excutables) as well as using the debugger. I also created custom test cases to test edge cases.

ML Forum Post Classifer (Class Project)

This project is also a C++ command-line program that identifies subjects of posts from Piazza by applying natural language processing & machine learning techniques. Posts were categorized into relevant threads or subject-matter using binary search trees & recursion. Also developed Bayesian model to process textual data, enhancing platform functionality. This project helped hone in skills with working with a partner and data structures like trees and recursive techniques.

Self-Learning Chatbot

Created in Python, the program uses ML techniques to check if a question already in its local JSON database with a 0.8 similarity cutoff. If it doesn't then chatbot asks the answer and stores it. As more questions and answers are prompted by a user, chatbot develops more answers.
